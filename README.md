# CODSOFT-Tic-Tac-Toe
# Tic Tac Toe Game 🎮

This project is a simple **Tic Tac Toe game built using Python**.
The game allows two players to play Tic Tac Toe in the terminal by taking turns placing their marks (X and O) on a 3×3 grid.

---

## 📌 Project Overview

Tic Tac Toe is a classic two-player strategy game.
Players take turns marking spaces in a **3×3 grid**. The player who succeeds in placing **three of their marks in a horizontal, vertical, or diagonal row wins the game**.

This project demonstrates:

* Basic Python programming
* Game logic implementation
* Conditional statements and loops
